
# RTR AI Real-Time Payments Monitoring Dashboard

AI-powered simulation of Canada's Real-Time Rail (RTR) transaction monitoring.

## Features
- Real-time payment simulation
- AI anomaly detection (Isolation Forest)
- Bank liquidity tracking
- Transaction risk scoring
- Interactive dashboard using Dash + Plotly

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Open browser:

http://127.0.0.1:8050
